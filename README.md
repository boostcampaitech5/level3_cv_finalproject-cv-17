#### 프로젝트 소개
현대인의 스마트폰 사용 시간이 지속적으로 증가하는 주세에 있어, 스마트폰을 보다 편하게 조작하기 위한 기능을 개발하고자 한다. 
이를 통해 몸이 불편하거나, 자세가 고정된 상황에서도 스마트폰을 사용이 가능하도록 한다.
